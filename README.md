# SIFRAH - Plataforma Interna

Plataforma interna de SIFRAH construida con Next.js y TypeScript.

## Características

- ✅ Diseño responsive (Desktop y Mobile)
- ✅ Página de Login
- ✅ Página de Registro
- ✅ Diseño moderno con gradientes y animaciones
- ✅ Validación de formularios
- ✅ Integración con iconos sociales

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Estructura del Proyecto

```
app/
  ├── components/
  │   ├── BrandingPanel.tsx    # Panel de branding con gradiente
  │   └── SocialIcons.tsx      # Iconos de redes sociales
  ├── login/
  │   └── page.tsx             # Página de login
  ├── registro/
  │   └── page.tsx             # Página de registro
  ├── layout.tsx               # Layout principal
  ├── page.tsx                 # Página principal (redirige a login)
  └── globals.css              # Estilos globales
```

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- React Icons
